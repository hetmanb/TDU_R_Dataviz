Chapter 2 Training Datasets for use in the course

This chapter will include references to openly available datasets and provide some suggestions for how we can use them.

2.1 Summary of datasets

The following tables provide links to data useful for either public-health based examples, or other fun examples for use. Note that some of these data sets are internal to the TDU and require internal permissions to access. For more information, or if you would like to gain access to an internal training data set, please contact the TDU.
